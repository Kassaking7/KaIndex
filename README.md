# KaIndex

This is a Web Terminal developed using Vue3, Vite, Pinia, Ant Design and Express.js
Developed features like searching and bookmarking


### Frontend Tech Stack
#### Techs
* Vue3
* Ant Design Vue 3
* Pinia 2
* TypeScript
* Eslint
* Prettier
#### Library
* Axios
* DayJs
* Lodash
* Getopts

### Backend Tech Stack
#### Techs
* Node.js
* Express.js
* MySQL
* Redis
